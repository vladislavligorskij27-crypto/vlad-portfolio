🚀 Personal Portfolio & Software Engineer Showcase

Добро пожаловать в мой центральный проект! Это современный, высокопроизводительный сайт-портфолио, созданный для демонстрации моих навыков в системном программировании, веб-технологиях и архитектуре баз данных.

🛠 Технологический стек (Tech Stack)

Languages & Core

Java: Разработка надежных серверных решений (Spring Boot).

C++ / C#: Системное программирование и высоконагруженные приложения.

Python: Скрипты автоматизации, парсинг данных и аналитика.

SQL: Проектирование и оптимизация сложных реляционных баз данных (PostgreSQL, MySQL).

Frontend & UI/UX

HTML5 & ES6+ JavaScript: Динамика и логика на стороне клиента.

Tailwind CSS: Современная верстка с использованием утилитных классов.

Lucide Icons & FontAwesome: Профессиональная визуализация интерфейса.

✨ Ключевые особенности

🌓 Адаптивная тема (Dark/Light): Интеллектуальное переключение цветовой схемы с сохранением состояния пользователя.

🤖 Интеграция с Telegram API: Реализована "бессерверная" (Serverless) отправка сообщений из формы обратной связи напрямую в мессенджер разработчика.

📈 Динамический таймлайн: Визуализация пути развития навыков и ключевых этапов обучения.

⚡ Оптимизация производительности: Минимальный вес страницы, отсутствие тяжелых зависимостей и быстрая загрузка.

🚀 Как работает обратная связь?

В проекте реализован прямой мост между браузером пользователя и Telegram Bot API:

Сбор данных: JavaScript формирует объект из полей ввода.

Запрос: Используется асинхронный метод fetch для отправки POST-запроса.

Доставка: Бот мгновенно пересылает сообщение владельцу репозитория, обеспечивая связь 24/7.
