О проекте

Vlad_Portfolio — это не просто статичная страница, а полноценная визитка инженера, спроектированная с упором на производительность и UX. Сайт служит центральным узлом для демонстрации стека (Java, C++, C#, Python) и реализованных кейсов.

Основной упор сделан на Zero-Backend архитектуру: отправка данных из формы обратной связи происходит напрямую через Telegram Bot API, что исключает необходимость в промежуточном сервере и обеспечивает мгновенную доставку уведомлений владельцу.

Ключевой функционал

Интеллектуальный UI: Динамическое переключение тем (Dark/Light) с сохранением предпочтений в localStorage.

Typewriter Engine: Кастомная реализация эффекта печатающейся машинки для презентации ключевых тезисов.

Direct-to-Telegram: Прямая интеграция с мессенджером для получения заявок в режиме реального времени.

Адаптивность: Полная поддержка всех типов устройств (от смартфонов до 4K мониторов) благодаря Mobile-First подходу.

Интерактивный таймлайн: Визуализация этапов обучения и профессионального роста с использованием анимаций наведения.

Техническая реализация и Архитектура

Frontend Стек: Использование Vanilla JavaScript (ES6+), HTML5 и Tailwind CSS для минимизации веса страницы.

Событийная модель: Применение Intersection Observer API для реализации "ленивой" анимации появления элементов при скролле (Fade-in эффекты).

Сетевое взаимодействие: Использование асинхронного метода fetch для взаимодействия с Telegram Bot API. Данные передаются в формате JSON по протоколу HTTPS.

Оптимизация рендеринга: Использование кастомных CSS-фильтров (backdrop-blur) для создания эффекта матового стекла в навигации без ущерба для скорости отрисовки.

State Management: Состояние темы и выбор пользователя управляются нативно через браузерное хранилище данных.

Установка и запуск (Local Setup)

1. Клонирование репозитория

git clone [https://github.com/vladislavligorskij27-crypto/vlad-portfolio.git](https://github.com/vladislavligorskij27-crypto/vlad-portfolio.git)
cd vlad-portfolio


2. Настройка конфигурации

Для работы формы обратной связи откройте index.html и укажите данные вашего бота в секции JavaScript:

const TELEGRAM_BOT_TOKEN = 'ВАШ_ТОКЕН';
const TELEGRAM_CHAT_ID = 'ВАШ_ID';


3. Запуск

Просто откройте файл index.html в любом современном браузере. Установка дополнительных зависимостей не требуется.
