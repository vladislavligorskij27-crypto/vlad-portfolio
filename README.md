✨ О проекте

Этот репозиторий содержит исходный код моего личного сайта-портфолио. Главная цель проекта — показать навыки написания чистого кода, внимание к деталям (UI/UX) и умение создавать нестандартные интерактивные решения. Сайт написан на Vanilla JS без использования тяжелых фреймворков (типа React или Vue), что обеспечивает максимальную скорость работы, отсутствие лишних зависимостей и плавность всех анимаций.

🔥 Ключевые особенности

💻 Встроенный Терминал: Нажмите клавишу ~ (тильда) или иконку консоли, чтобы открыть стилизованный терминал. Он поддерживает:

Автодополнение команд (клавиша Tab)

Историю ввода (стрелочки вверх/вниз)

Кастомные команды (whoami, help, echo и др.)

🎮 Мини-игры и Пасхалки: * Введите команду play snake в терминале для запуска классической змейки прямо поверх консоли.

Команды matrix, stress-test, destroy запускают особые визуальные режимы (например, мини-игру по расстрелу элементов сайта).

Вызов легендарной мобильной "Бочки" (Barrel Roll) при клике на консоль с телефона.

Пятикратный клик по логотипу активирует секретный "Режим разработчика" 🪩.

🌓 Темизация и Локализация: Мгновенное переключение между светлой/темной темой оформления и русским/английским языком. Состояние бережно сохраняется в localStorage.

🔗 Интеграция с GitHub API: Раздел "Мои проекты" и блок статистики автоматически подтягивают актуальные данные (звезды, форки, количество подписчиков) напрямую из моего профиля в реальном времени.

📱 Адаптивный дизайн: Полностью отзывчивый интерфейс, который подстраивается под мобильные устройства (включая кастомное мобильное меню-гамбургер).

🚀 Как запустить локально

Для запуска проекта на вашем компьютере не требуются сборщики вроде npm или webpack. Всё максимально просто:

# 1. Склонируйте репозиторий
git clone [https://github.com/vladislavligorskij27-crypto/vlad-portfolio.git](https://github.com/vladislavligorskij27-crypto/vlad-portfolio.git)

# 2. Перейдите в папку с проектом
cd vlad-portfolio

# 3. Откройте файл index.html в любом браузере
# (Например, просто перетащите файл index.html в окно Chrome или Safari)
